Ques: How does javascript work?

Ans:
Well, JavaScript is a scripting language.It's use in a web-page to make data updating,control multimedia and other lots of works.Javascript is a scripting language.Specially, it is a client-side scripting language.It's means it works in client machine.Every web-browser has a default javascript engine.For example, Google Chrome uses V8 engine.

JavaScript is an interpreted language.When a browser loads a web-page,the browser has a built-in interpreter thats reads the javascript code line by line (By default) and runs it.By default it reads and loads code asynchronously.That means it happen independently of the main program flow.Javascript also works on callback function.It is a function that passing a value to another function and it executed when the event happens.

Ques: Difference between local storage vs session storage?

Ans:
There are two kinds Web Storage API .One is localStorage and another is sessionStorage.This two type storage maintain different storage area of the web-page session.Both of these have a limit of 5MB storage space.

The main difference between two storage system is that sessionStorage only maintain a storage area while the web-browser is open.Even it can store when web-pages reload or restores.Page session starts when the web-browser is open. Besides, the data of sessionStorage is cleared when the web-page is closed.
On the otherhand,localStorage can continue store data after the web-browser is closed.So that data of localStorage does not expire like sessionStorage.
